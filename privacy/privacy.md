# Privacy Policy

**Last Updated:** 11/04/2026

## 1. Introduction

Welcome to Bicly.

We are committed to protecting your privacy and minimizing the amount of personal data we process.

We do not create user accounts, set tracking cookies, build user profiles, or use analytics tools. Our goal is to operate a privacy‑respecting service with the smallest possible data footprint.

## 2. Data We Collect

The only personal data we process consists of automatic server access logs generated by our infrastructure (e.g., nginx, brouter).

### 2.1 Server Log Data

When you access our website, our servers automatically record:

- IP address
- Date and time of the request
- Requested URL
- User‑Agent string
- Referrer header (if provided)

### 2.2 Purpose of Processing

We process this data solely to:

- Ensure the security and stability of our service
- Detect and prevent abuse (e.g., DDoS, scraping, attacks)
- Diagnose technical issues
- Generate non‑personal, aggregated traffic statistics

We do not use this data for marketing, profiling, or tracking.

### 2.3 Legal Basis (GDPR)

Processing is based on **Legitimate Interests** (Article 6(1)(f) GDPR), specifically:

- Operating a secure and reliable website
- Protecting our infrastructure
- Ensuring service availability

## 3. Third‑Party Services (Independent Controllers)

Our website integrates mapping and geocoding features. These features operate via direct API calls from your browser to external providers.

This means:

- Your device connects directly to the third‑party service
- Your IP address and request details are sent without passing through our servers
- We do not receive, store, or access this data

Under GDPR, these providers act as **Independent Data Controllers**.

### 3.1 OpenStreetMap Nominatim (Geocoding)

| Property | Details |
| :--- | :--- |
| **Purpose** | Converts place names into geographic coordinates |
| **Data Flow** | Your browser sends search queries and your IP address directly to `https://nominatim.openstreetmap.org` |
| **Our Role** | We do not log or store your geocoding queries |
| **Privacy Policy** | [https://osmfoundation.org/wiki/Privacy_Policy](https://osmfoundation.org/wiki/Privacy_Policy) |
| **Rights Requests** | Must be directed to the OpenStreetMap Foundation |

### 3.2 OpenFreeMap (Map Tiles)

| Property | Details |
| :--- | :--- |
| **Purpose** | Provides map tiles and vector data for map display |
| **Data Flow** | Your browser requests map tiles directly from `https://openfreemap.org`, revealing your IP address and the tile coordinates of the map area you view |
| **Our Role** | We do not track or store which map areas you view |
| **Privacy Policy** | [https://openfreemap.org/privacy/](https://openfreemap.org/privacy/) |
| **Rights Requests** | Must be directed to OpenFreeMap |

## 5. Your Rights Under GDPR

Because server logs may contain your IP address, you have the following rights regarding the data we control:

| Right | Description |
| :--- | :--- |
| **Right of Access** | Request a copy of log entries associated with your IP address |
| **Right to Erasure** | Request deletion of your log data |
| **Right to Restriction** | Request limited processing |
| **Right to Object** | Object to processing based on legitimate interests |
| **Right to Data Portability** | Receive your log data in a structured format (e.g., CSV) |

To exercise these rights, you may need to provide:

- The IP address you used
- The approximate date/time of access
